# Thai Pulp Packaging - Website (Video Background Slider)

## โครงสร้างไฟล์
- index.html
- style.css
- script.js
- /img  (ใส่รูปโลโก้และรูปต่างๆ)
- /media (ใส่วิดีโอพื้นหลัง 5 ไฟล์)

## 1) ใส่วิดีโอ 5 ไฟล์
นำไฟล์ mp4 ของคุณใส่ในโฟลเดอร์ `media/` และตั้งชื่อให้ตรง:
- hero1.mp4
- hero2.mp4
- hero3.mp4
- hero4.mp4
- hero5.mp4

(ถ้าตั้งชื่อไม่ตรง ให้แก้ใน index.html ที่ <source src="media/heroX.mp4">)

## 2) ใส่รูป (โลโก้/สินค้า/แกลเลอรี)
ใส่ไฟล์รูปในโฟลเดอร์ `img/` ตามชื่อที่ใช้ใน index.html เช่น:
- img/logo.png
- img/about.jpg
- img/product_pot.jpg, img/product_cup2.jpg, img/product_cup3.jpg, img/product_custom.jpg
- img/gallery1.jpg ... img/gallery6.jpg

## 3) เปิดแบบ Live Server (แนะนำ)
VS Code -> ติดตั้ง Live Server -> คลิกขวา index.html -> Open with Live Server
